# 2022 2학기 객체지향프로그래밍(자바 프로젝트)

### 참고사항
- MainMenu 파일의 key값에 공공데이터포털의 키값을 넘겨주어야 코로나 확진자 수를 알려줌
- Client1To1EndToEnd 파일과 Server1To1EndToEnd 파일속 존재하는 callApi메소드의 키값을 본인의 표준국어대사전 인증키값에 맞게 수정해야함

### 개발버전 및 추가라이브러리
JavaSE-18 사용 / macOS 13.0.1(22A400) / 이클립스 Version: 2022-06 (4.24.0) / json simple 라이브러리 필요

### 프로그램 오류 모음
- Api호출 실패시 따로 에러메시지 출력이 불가능함
- 그룹채팅시 엔터키를 눌러 입장하지 않고 나가면 null값 처럼 이상한 값을 넘기고 종료하므로 누가 나갔는지 정확히 알려줄 수 없음
- 끝말잇기 두음법칙 불가능
- 끝말잇기에서 단순히 상대측에서 보낸 마지막 글자와 내 첫번째 글자를 비교하기 때문에 상대측에서 아무글자나 넘겨서 내쪽으로 " 상대측에서 검색할 수 없는 단어 [ㅁㅇㄴㄹ]를 입력하였습니다. " 라고 뜨게 된다면 마지막 문자인 [ . ] 과 비교한다. 때문에 서로 주고받는 순서가 조금이라도 틀리면 종료시키고 다시 시작해야 한다. 대신 내 차례에서 다른 글자를 입력한 경우는 이미 상대측에서 넘긴 글자가 유지되고 있기 때문에 상관 없다.
